    • viewer/manager/SearchHighlighter

    Class default

    The SearchHighlighter module performs searches over the indexed text, then applies inline highlights to the corresponding text layer elements. It also maintains a flat list of highlighted elements for result navigation.

    Integration Note: Each page container should have an ID in the format "pageContainer-{pageNumber}" and its text layer should use the class "a-text-layer". The text spans inside the layer should have the attribute role="presentation".
