Class default

SearchBar creates the search UI and wires events to perform a search, navigate matches, and update the match counter.

It expects the following instance methods to be available:

  • this.search(searchTerm: string, options: { matchCase: boolean, regex: boolean, wholeWord: boolean })
  • this.prevMatch()
  • this.nextMatch()

It also stores references to its UI elements (search input, match counter, etc.) for further use (like updating the counter).
